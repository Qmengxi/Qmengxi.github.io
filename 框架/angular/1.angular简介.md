# Angular 学习笔记

## 1. angular 简介：

​	Angular是Web前端开源框架，基于TypeScript开发，较react，Vue，angular更适合中大型企业级项目。

## 2. 环境搭建

### 1. 必备环境

> NodeJS

> npm/cnmp

> angular/cli 脚手架

`npm install -g @angular/cli`

## 3. 创建项目

### 1）创建项目

`ng new angulardemo`  

### 2) 安装依赖模块

`npm install`

### 3) 运行项目

`ng sever --open --port 4002`
